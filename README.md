## Google Spreadsheet

Using Roo to access Google spreadsheets requires you install the 'google-drive-ruby' gem separately.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'roo-google'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roo-google
